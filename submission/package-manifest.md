# Package Manifest

## Website ZIP

`linyuanxing-ncda-c1-site.zip` 应包含：

- 根目录 15 个 HTML 页面
- `style.css`
- `script.js`
- `favicon.svg`
- `assets/` 图片、logo 与视频资源
- `README.md`
- `docs/` 项目说明与检查文档

不包含：

- `.git/`
- `.vscode/`
- `submission/artifacts/`
- 本地临时文件
- 署名版身份材料

## Judge Route

1. `demo.html`：先看 3 分钟宣讲路径
2. `index.html`：进入首页首屏、PV、角色、地图、剧情与 Project
3. `archives.html`：验证世界观档案库与静态筛选
4. `characters.html`：进入四名角色详情页
5. `archive.html`：进入两份剧情档案详情页
6. `reserve.html` / `login.html`：确认静态占位边界
