# Anonymous Compliance Checklist

## Website

- [ ] 15 个 HTML 页面不出现学校、学院、专业班级、学号、作者姓名或指导老师姓名。
- [ ] 页面不使用赛事官方 logo、证书样式或官方宣传素材。
- [ ] `reserve.html` 与 `login.html` 明确为静态占位，不收集用户数据。
- [ ] 项目不宣称真实游戏已上线或真实运营。

## Visual Materials

- [ ] 作品简介图不出现学校、作者、指导老师或赛事官方标识。
- [ ] 60×80cm 展板不出现学校、作者、指导老师或赛事官方标识。
- [ ] A3 海报不出现学校、作者、指导老师或赛事官方标识。
- [ ] 展示截图不包含本地绝对路径、系统用户名或浏览器账号信息。

## Video

- [ ] 视频时长不超过 3 分钟。
- [ ] 视频封面和字幕不出现学校、作者、指导老师或赛事官方标识。
- [ ] 旁白只介绍作品，不透露个人身份信息。
- [ ] 视频明确说明作品为静态交互网页 MVP，不暗示真实登录、预约或后端服务。

## ZIP

- [ ] ZIP 解压后可直接打开 `index.html`。
- [ ] ZIP 内保留相对路径结构。
- [ ] ZIP 不包含 `.git`、`.vscode`、本机临时文件或个人身份材料。
- [ ] 匿名评审包与署名存档材料分开。
