# Stage 1A Notes

## 当前首页模块边界

- Hero：`index.html` 顶部首屏区域，包含品牌、主导航、首屏文案与首屏按钮。
- PV：`#pv`，保留现有 PV 展示与播放按钮交互。
- Characters：`#characters`，保留角色卡片、视频片段、展开内容和人物弹窗。
- Map / Scenes：`#map-trace`，保留场景卡片与灯箱预览。
- Archive：`#archive`，用于剧情档案与时间线展示。
- Lore：`#lore`，用于世界观卡片与折叠设定内容。
- Footer：`#site-footer`，用于站点尾部信息与未来入口占位。

## 已预留但尚未实现的区域

- `news.html`：未来新闻、公告、版本动态入口占位。
- `reserve.html`：未来预约落点占位。
- `login.html`：未来登录入口占位。
- 首页页脚下载入口文案：仅显示“敬请期待 / 预约开启”性质的静态提示，不提供真实下载能力。

## 当前阶段明确不要实现的内容

- 不做后端、数据库、账号体系、鉴权、后台管理。
- 不做预约提交、表单收集、短信/邮件回执、媒体上传。
- 不做 API 请求、不接真实新闻数据、不抽取 JSON 数据源。
- 不迁移到 React、Vue、Next.js、Vite 或其他框架。
- 不大改首页视觉、排版节奏、颜色气质、字体方向和现有交互。
