# LinYuanXing 临渊行

《临渊行》是一个国风幻想游戏宣发网站，用于展示游戏世界观、角色、场景、PV 与交互内容。

## Online Demo
Netlify 在线演示地址

## Features
- Cinematic landing page
- Worldbuilding and story sections
- Character cards and modal interactions
- Scene gallery
- PV presentation module
- Responsive static frontend

## AI-assisted Workflow
This project was built and iterated with ChatGPT/Codex-style AI workflow, including information architecture planning, copywriting refinement, frontend code editing, interaction checking, and deployment iteration.

## Tech Stack
HTML, CSS, JavaScript, Netlify, GitHub
