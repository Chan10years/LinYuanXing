# 临渊行

《临渊行》是一个偏国风志怪与叙事冒险气质的静态游戏官网概念项目，用于展示世界观、角色、场景、PV 与页面信息架构能力。

## Online Demo
- Netlify 在线地址：`https://linyuanxing-demo.netlify.app`

## 项目定位
- 面向作品集展示的概念游戏官网
- 重点体现世界观包装、页面节奏、视觉统一性与前端静态交互
- 当前不承载真实账号、预约、后台或数据系统

- Stage 2 进行中：已强化首页内容纵深、Project 作品集表达、地图寻迹文本与预约页静态展示细节
