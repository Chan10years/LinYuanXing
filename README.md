# 临渊行

《临渊行》是一个偏国风志怪与叙事冒险气质的静态游戏官网概念项目，用于展示世界观、角色、场景、PV 与页面信息架构能力。

## 项目定位
- 面向作品集展示的概念游戏官网
- 重点体现世界观包装、页面节奏、视觉统一性与前端静态交互
- 当前不承载真实账号、预约、后台或数据系统

## 已完成功能
- 首页 Hero 主视觉与主导航
- PV 展示与播放交互
- 角色卡片、片段展开与人物资料弹层
